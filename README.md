# Drone Monitor Map

Професійна інтерактивна карта для моніторингу повітряних загроз (БПЛА, Ракети) з інтеграцією Telegram.

## Можливості
- **Інтерактивна карта:**
  - Dark Mode стилізація.
  - Векторні мапи областей України (GeoJSON) з підтримкою підсвічування тривог (червона заливка).
  - Маркери загроз (БПЛА/Ракета) з напрямком руху та кількістю (групи).
  - Лінії траєкторії польоту.
- **Адмін-панель:**
  - Додавання загроз перетягуванням або кліком.
  - Групування загроз (вказівка кількості "x3").
  - Обертання напрямку руху (ПКМ - Права кнопка миші).
  - Керування статусом тривог по областях (клік по області).
  - Лічильник загальної кількості цілей.
- **Автоматизація:**
  - Генерація скріншотів високої якості (Playwright).
  - Автоматична відправка скріншоту в Telegram-канал/чат.
  - Збереження стану карти при перезавантаженні сторінки.

## Встановлення

### Вимоги
- Python 3.9+
- Node.js (опціонально, використовується Python Playwright)

### Кроки
1. **Клонувати або завантажити папку.**
2. **Налаштувати оточення:**
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   pip install -r requirements.txt
   playwright install chromium
   ```
3. **Налаштувати Telegram:**
   - Створіть файл `.env` на основі `.env.example`.
   - Вставте токен бота та ID чату.

### Запуск
```powershell
.\venv\Scripts\python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
Відкрити в браузері: [http://localhost:8000](http://localhost:8000)

## Використання
1. **Додавання загрози:** Оберіть тип (БПЛА/Ракета), введіть кількість, клікніть на карті.
2. **Напрямок:** Правою кнопкою миші по маркеру для повороту.
3. **Тривога:** Клікніть по області України, щоб увімкнути/вимкнути червоний режим.
4. **Скріншот:** Натисніть кнопку "Зробити скріншот". Бот відправить фото в канал.

## API (для автоматизації)
- `GET /api/state` - Отримати поточні загрози та тривоги.
- `POST /api/state` - Оновити список загроз (JSON).
- `POST /api/screenshot` - Згенерувати та відправити скріншот.
